Инструкция по подготовке файла

Ваш файл должен быть в одном из следующих форматов: CSV, XLS, XLSX.

Каждая колонка файла должна содержать слова или фразы, которые вы хотите использовать для перемножения.

Пример:

Колонка 1 Колонка 2 Колонка 3
слово1 слово2 слово3
слово4 слово5 слово6
